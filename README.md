## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm
- MongoDB instance (local or Atlas)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Halyna13890/Instagram_project.git
    cd instagram-project
    ```

2.  **Backend Setup**
    ```bash
    cd backend
    npm install
    ```

    - Create a `.env` file in the `backend` folder and add:
        ```
        PORT=5000
        MONGO_URI= (Coming soon)
        JWT_SECRET= (Coming soon)
        ```

    - Run the backend server:
        ```bash
        npm run dev
        ```

3.  **Frontend Setup**
    ```bash
    cd ../frontend
    npm install
    npm start
    ```
    The frontend application will open at `http://localhost:3000`.
